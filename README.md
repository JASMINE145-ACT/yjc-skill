# yjc-skill

Cursor / Neon 业务 Skill 合集，可供 Cursor 或其它 LLM 直接调用。

## 包含内容

- **Neon 四大 Skill**：inventory-price-query、oos-shortage-register（见 `plan-code-review/`）、quotation-register-from-dialog、replenishment-register
- **其它**：plan-code-review、pptx、skill-creator、wanding-select、webapp-testing

## 使用与分发

详见 [README_分发与下载.md](README_分发与下载.md)：打包方式、对方使用前提、各 Skill 的 SKILL.md 说明。

## 克隆后放在 Cursor 中

将本仓库克隆或解压到项目的 `.cursor/skills/` 下，或按 Cursor 约定放到用户级 skills 目录；也可只复制需要的单个 skill 目录使用。
